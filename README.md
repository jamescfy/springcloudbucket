# springcloudbucket
a family bucket for springcloud learner

快速开始
如果你还没有克隆项目。
git clone https://github.com/jamescfy/springcloudbucket.git
如果需要，可以从application.yml文件更改端口号。
eg:server.port=80
您也可以在Eclipes或IntelliJ IDEA中运行它。找到程序主入口并运行它。
